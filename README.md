# Projeto de Carreira

Trabalho de planejamento de carreira e sucesso profissional da Unasp, desenvolvido como um site pessoal estático em formato de portfólio, currículo online e plano de carreira de Pedro Henrique Menardi Ferraz de Almeida.

## Arquivos principais

- `index.html`: conteúdo completo do portfólio e plano de carreira.
- `style.css`: estilo minimalista em preto, branco e cinza.
- `script.js`: menu mobile, scroll spy, validação do formulário e timeline interativa.

## Publicar no GitHub Pages

1. No repositório, acesse `Settings` > `Pages`.
2. Em `Build and deployment`, selecione `Deploy from a branch`.
3. Escolha a branch `main` e a pasta `/root`.
4. Salve e aguarde o link público ser gerado.

## Configurar Formspree

1. Crie uma conta gratuita em [Formspree](https://formspree.io/).
2. Crie um formulário e copie o ID fornecido.
3. No arquivo `index.html`, substitua `https://formspree.io/f/SEU_ID_AQUI` pelo endereço real do formulário.

## Currículo em PDF

O botão de download aponta para `cv-pedro-menardi.pdf`. Para ativá-lo, coloque o arquivo PDF com esse nome na raiz do projeto.
